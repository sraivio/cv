About this page

This CV project is an unholy union of AI and my own skills. The code is pretty much vibed, but the (easy) decision to host in GitHub Pages as SPA, UX decisions, and design direction are intentionally mine.

The page is still very much a work in progress.

Currently, the layout responds only to the season and time of day in Helsinki. Next steps on the roadmap are:

  • Integrating a weather API and dynamically adapting the UI to real-time conditions in Helsinki. The meteorology-inspired theme reflects an ongoing personal interest, which you can find also mentioned in the Experience section if you read it carefully.
  • Enhanching accessibility features. Now the page should be fairly usable for all device sizes, but I acknowledge that the Skills cannot be copied, the keyboard navigation is not perfect and I haven't paid attention to any screenreader support or things like that.
